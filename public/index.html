Deployment Runbook

Ship locally, on Pharos, and on Vercel.

  1. npm install
  2. cp .env.example .env
  3. Add your funded Pharos Atlantic PRIVATE_KEY.
  4. npm run compile
  5. npm test
  6. npm run deploy
  7. npm run demo:server
  8. npm run demo:x402
  9. npm run demo:agentic
  10. npx vercel --prod
